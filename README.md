# TripleGGGCardComp
TripleGGGCardComp

This repository now includes a static GitHub Pages site for tracking card comps and sending the current card set to multiple comps destinations.

## Features
- Add cards with comp prices and keep them in local storage.
- Calculate the average comp price from the tracked cards instantly.
- Select multiple comps destinations and open the current card set in each selected site.

## Run locally
- Serve the repository root with a static server, for example:
  - `python3 -m http.server 8000`
- Open `http://127.0.0.1:8000/` in a browser.
